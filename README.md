## ğŸ“ How to Submit Feedback

### Reporting a Bug

If you've encountered a bug or issue:

1. **Check existing issues** - Search the [issue tracker](https://github.com/space-launch-now/feedback/issues) to see if the bug has already been reported
2. **Create a new issue** - Click on the "New Issue" button
3. **Provide details** - Include:
   - Clear description of the bug
   - Steps to reproduce
   - Expected vs actual behavior
   - Device/platform information (OS, app version, etc.)
   - Screenshots or screen recordings (if applicable)

### Suggesting a Feature

Have an idea for a new feature or improvement?

1. **Check existing issues** - See if someone has already suggested something similar
2. **Create a new issue** - Use the "New Issue" button
3. **Describe your suggestion** - Include:
   - Clear description of the feature
   - Use case or problem it solves
   - Any relevant examples or mockups

### General Feedback

For general feedback, comments, or questions:

1. Create an issue with your feedback
2. Use appropriate labels to categorize your submission
3. Be respectful and constructive in your communication
